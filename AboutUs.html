We have implemented following features in our WebApp:


Astronomy Picture of the Day: NASA Open APIs make available NASA data, including imagery, eminently accessible to software developers. One of the most popular NASA Open APIs is the Astronomy Picture of the Day'(APOD). It structures the APOD imagery and associated metadata so that it can be repurposed for other applications

Currency conversion: Provides currency exchange rates based on the markets and national central banks data.

Weather Forecast: Weather Forecast displayed using the Open Weather API. It allows you to gather
weather data for over 200,000 cities.You can also use the API to get access to historical weather data.The data collection includes
Current Weather Data, Hourly Weather Data 4 Days, Daily forecast 16 days, Climatic Forecast 30 days, etc.

Countries Information: Shows information of the selected country. Implemented with the REST Countries API which is a RESTful API that allows you to get information about the world's countries.
For example, you can send a query to get a country's language, currency, and capital.

News API: We show news from around the world. This is enabled with the News API , a simple and easy-to-use API that returns JSON-formatted news from a variety of sources.



Architectrue and Soltion Highlights of our WebApp:


Fully Decoupled Architecture: User interface layer fully decoupled and independent with backend layer. So any change in one layer or component (User Interface, Backend APIs) will not impact code in other layer.

Digital Backend: Digital backend (built with independent APIs) layer (component), since composed of multiple independent APIs ensures no single point of failure (if there is issue one API other functionality keeps working).

Globally Available API: It enables using API as building blocks and avoid reinventing the wheel. Not spent time in developing APIs for functionality which is already available saves time and cost.

Rapid Development: Solution approach enables rapid development which reduces time of development and faster launch of various functionalities.
